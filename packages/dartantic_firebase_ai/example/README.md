# Firebase AI Provider Example

Flutter example app demonstrating the Firebase AI provider with Dartantic.

## Setup

1. **Configure Firebase**:
   ```bash
   # Install FlutterFire CLI
   dart pub global activate flutterfire_cli

   # Configure Firebase for your project
   flutterfire configure
   ```

2. **Install dependencies**:
   ```bash
   flutter pub get
   ```

3. **Enable Firebase AI** in your Firebase console:
   - Go to your Firebase project console
   - Navigate to "Build" > "AI Logic"
   - Enable Firebase AI Logic

## Running

```bash
flutter run
```

## Code Structure

- `lib/main.dart` - Simple chat UI using `FirebaseAIProvider` and `ChatModel` directly, with streaming responses and conversation history.
- `lib/all_systems_go.dart` - Smoke test that verifies Firebase initialization and provider registration.
- `lib/firebase_options.dart` - Firebase project configuration (generated by FlutterFire CLI).

## Notes

- This is a Flutter app (not pure Dart) because `firebase_ai` and `firebase_core` require the Flutter SDK.
- Firebase must be configured for your project via `flutterfire configure`.
- Firebase AI Logic must be enabled in your Firebase console.
